feGaussianBlur 應該算是 SVG filter 裏頭最簡單的濾鏡，
因為它只有一個參數需要注意：stdDeviation，這也是控制模糊程度的參數，
數字越大越模糊，數字為零則是圖片原本的狀態，在這邊有一個比較需要注意的地方，
就是要記得設定 filter 的 filterUnits 為 userSpaceOnUse ，
因為預設值是會讓 filter 跟隨著套用該 filter 的物件跑，
所以要記得設定成跟隨整個 SVG space。
